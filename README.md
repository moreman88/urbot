# Urbot

Локальный Telegram-бот для автоматизации учебной части.

Важно: **Никогда не храните действительный токен в исходном коде или в публичных чатах.** Если токен был раскрыт — немедленно отозовите его в @BotFather и создайте новый.

## Установка (Windows / PowerShell)

1) Установите зависимости в виртуальном окружении:

```powershell
python -m venv .venv
.\.venv\Scripts\Activate.ps1
pip install -r requirements.txt
```

2) Установите токен (вариант A — временно в текущей сессии):

```powershell
$env:URBOT_BOT_TOKEN = 'ВАШ_ТОКЕН_ЗДЕСЬ'
python -u 'C:\Users\Sanat\Desktop\Urbot\Urbot.py'
```

Вариант B — навсегда для пользователя (перезапустите терминал после setx):

```powershell
setx URBOT_BOT_TOKEN 'ВАШ_ТОКЕН_ЗДЕСЬ'
# затем откройте новый PowerShell и запустите
python -u 'C:\Users\Sanat\Desktop\Urbot\Urbot.py'
```

3) Альтернатива — создайте локальный `.env` файл рядом с `Urbot.py` (НЕ добавляйте `.env` в репозиторий):

```
# .env
URBOT_BOT_TOKEN=ваш_токен_здесь
```

Скрипт автоматически загрузит `.env` (через `python-dotenv`).

## Запуск

```powershell
python -u 'C:\Users\Sanat\Desktop\Urbot\Urbot.py'
```

## Безопасность

- Если вы случайно опубликовали токен — обязательно отзовите его в BotFather и сгенерируйте новый.
- Не коммитьте `.env` в репозиторий. Добавлен `.gitignore`, чтобы избежать этого.
